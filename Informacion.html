En el formulario de Socio.js contiene lo que son las URL de Postman de las API ya creadas anteriormente haciendo uso de AJAX/JQuery de las 5 API Rest para el consumo, también cree 5 funciones que son Cargar Socios, Cargar socio, Agregar socio, Actualizar socio y Eliminar socio las cuales son las que nos permiten para que se carguen los datos y se pueda actualizar editando cada uno de los socios encontrados así mismo se pueda agregar un nuevo socio y también eliminando cualquiera de los ya creados.
